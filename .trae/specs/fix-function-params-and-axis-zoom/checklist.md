# Checklist

## 函数系统参数面板 UX
- [x] 输入 `y=x^2`（无参数）后，参数面板可见，显示提示文本和"添加参数"按钮
- [x] 提示文本明确告知用户如何使用参数功能（输入含参数表达式或手动添加）
- [x] 输入 `y=a*x^2+b`（含参数）后，参数面板显示 a, b 滑动条，无提示文本
- [x] "添加参数"按钮在含参数时也可显示（允许追加更多参数）

## 手动添加自定义参数
- [x] 点击"添加参数"按钮弹出输入框
- [x] 输入合法参数名（如 `k`）后，参数列表新增 `k` 滑动条（默认 -10~10, step 0.1, value 1）
- [x] 输入 `y=k*x` 后，拖动 `k` 滑动条图像实时变化
- [x] 即使表达式未使用自定义参数（如 `y=x^2` + 添加 `k`），也不报错
- [x] 输入已存在的参数名（如 `a` 已存在）显示错误提示，不重复添加
- [x] 输入保留字（x, pi, e）显示错误提示

## 参数识别 bug 排查

- [x] 添加函数后 `renderParamSliders()` 被立即调用
- [x] 3D 模式下参数面板正常工作
- [x] 参数滑动条拖动实时更新图像
- [x] 参数 min/max/step 设置生效
